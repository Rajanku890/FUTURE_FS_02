💼 Mini CRM – Client Lead Management System
📌 Project Overview

Mini CRM is a full-stack web application designed to manage client leads efficiently. It allows businesses to track leads from initial contact to conversion, manage follow-ups, and maintain communication history.

🚀 Tech Stack
Frontend
React.js
HTML5 / CSS3
JavaScript (ES6+)
Axios
Backend
Node.js
Express.js
Database
MongoDB (Mongoose)
Authentication
JWT (JSON Web Token)
